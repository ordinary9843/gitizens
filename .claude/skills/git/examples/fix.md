# fix — examples

Use when correcting a bug or broken behavior.

## Good

```
fix: clamp pollution delta to valid range 0–100
fix: skip expired events when no active event exists
fix: prevent double-counting votes from same user
fix: handle missing world/state.json on first run
```

## Bad

```
fix: fixed the bug              ← past tense, vague
fix: Bug fix                    ← not lowercase, not imperative
fix: various fixes              ← too vague
```
