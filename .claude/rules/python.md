# Python Source Rules

## Language

All code in this file must use English — variable names, comments, print statements, docstrings, error messages, and log output. No non-English text in source code. JSON game flavor text loaded from data files is exempt.

## Emoji

No emoji in Python source code. This includes:
- print statements and log output
- code comments (inline and block)
- docstrings
- string constants used as system/log messages

Permitted only in strings that are written directly into GitHub issue or PR bodies as end-user-facing voting instructions (e.g., 👍/👎 reaction prompts).
