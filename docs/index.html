WHAT IS GITIZENS

A civilization that lives inside a GitHub repo. Citizens propose laws as GitHub Issues, vote with 👍 reactions, and watch their world evolve in real time.
The world runs on its own every 4 hours — random events strike, eras rise and fall, buildings emerge from policy. No one controls it. Everyone shapes it.
